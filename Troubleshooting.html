On your first build, you will encounter computer issues, either due to improper installation or a faulty manufacturer. When you turn on the computer, the motherboard will tell you what appears to be wrong. It is critical to read the motherboard manual.

img { display: block; margin: auto; }

Yellow/Amber: Shows that there may be a problem with the DRAM and that you should be careful.

Red: An error or problem with the DRAM has been found.

CPU color codes:

Green: This means that the CPU is working properly.

Yellow/Amber: Shows that there may be a problem with the CPU and that you should be careful.

Red: An error or problem with the CPU has been found.

Boot color codes:

Green: This means that the boot process is going as it should.

Yellow and amber lights mean that there might be a problem with the boot process.

Red: An error or problem with the boot process has been found.

VGA color codes:

Green: This color means that the VGA or graphics card is working properly.

Yellow and amber lights mean that there could be a problem with the VGA or graphics card.

Red: A problem with the VGA or graphics card has been found.
